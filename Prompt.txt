STEP 1 — SMART CONTRACT: contracts/MicroLend.sol
═══════════════════════════════════════════════════════════════

Create MicroLend.sol with:

STRUCTS:
- LoanRequest: { uint id, address borrower, uint amount, uint creditScore,
  uint interestRate, uint duration, string purpose, LoanStatus status,
  uint createdAt, uint repayBy, bool repaid }
- LoanStatus enum: { PENDING, FUNDED, REPAID, DEFAULTED }

STATE VARIABLES:
- mapping(address => LoanRequest[]) public borrowerLoans
- mapping(uint => address) public loanIdToLender
- uint public totalLoansIssued
- uint public totalValueLocked
- uint public loanCounter
- address public owner
- uint public PROTOCOL_FEE = 5 (0.5% in basis points)

FUNCTIONS:
1. applyForLoan(uint amount, uint creditScore, string purpose, uint duration)
   - Require creditScore >= 600
   - Require amount between 0.001 ether and 1 ether
   - Require duration 30, 60, or 90 days
   - Set interestRate = 8 (8%)
   - Status = PENDING
   - Emit LoanApplied(loanId, borrower, amount, creditScore)

2. fundLoan(address borrower, uint loanId) payable
   - Require msg.value == loan.amount
   - Require loan.status == PENDING
   - Transfer ETH to borrower
   - Set loanIdToLender[loanId] = msg.sender
   - Set status = FUNDED
   - Set repayBy = block.timestamp + duration days
   - totalValueLocked += amount
   - Emit LoanFunded(loanId, lender, borrower, amount)

3. repayLoan(uint loanId) payable
   - Calculate repayment = principal + 8% interest
   - Require msg.value >= repayment
   - Transfer principal to lender
   - Transfer 0.5% protocol fee to owner
   - Set status = REPAID, repaid = true
   - totalValueLocked -= amount
   - Emit LoanRepaid(loanId, borrower, amount)

4. getLoansByBorrower(address borrower) returns LoanRequest[]
5. getAllPendingLoans() returns LoanRequest[] (loop all loanCounter IDs)
6. getProtocolStats() returns (totalLoansIssued, totalValueLocked)

MODIFIERS: onlyOwner, validLoanId

EVENTS: LoanApplied, LoanFunded, LoanRepaid, LoanDefaulted

Use Solidity ^0.8.0. Add full NatSpec comments on all functions.
Add ReentrancyGuard pattern manually (no imports needed).
Follow checks-effects-interactions pattern in all payable functions.

STEP 3 — BACKEND: backend/server.js + routes/creditScore.js
═══════════════════════════════════════════════════════════════

Express server on port 5000 with CORS enabled.

POST /api/credit-score endpoint that:
1. Accepts body: { income, employmentType, existingDebt, loanPurpose,
   repaymentHistory, utilityPayments, yearsAtAddress }

2. Calls OpenAI GPT-4o API with this exact system prompt:
   "You are a financial credit scoring AI specializing in alternative credit
   assessment for unbanked individuals in India. You evaluate creditworthiness
   using non-traditional data points instead of formal credit history.
   Analyze the provided data and return ONLY a valid JSON object with no
   additional text, markdown, or explanation. The JSON must have exactly
   these fields:
   {
     'score': <integer between 300 and 900>,
     'risk': <'low' | 'medium' | 'high'>,
     'approved': <true if score >= 600, else false>,
     'reason': <string, 2-3 sentences explaining the score in simple English>,
     'maxLoanAmount': <recommended max loan in ETH, between 0.001 and 1.0>,
     'suggestedRate': <interest rate percentage, always 8>
   }"

3. User message format to send:
   "Income: ₹{income}/month, Employment: {employmentType},
   Existing Debt: {existingDebt}, Loan Purpose: {loanPurpose},
   Past Repayment: {repaymentHistory}, Utility Bills: {utilityPayments},
   Residential Stability: {yearsAtAddress} years at current address.
   Generate a credit score."

4. Parse the JSON response and return it to the client.
5. If OpenAI fails, return a mock score object for demo mode.

Add error handling with try/catch. Log all requests.

Functions that call the smart contract:

1. applyForLoan(contract, signer, amount, creditScore, purpose, duration)

   - Wait for tx.wait()
   - Return { success, txHash, loanId }

   - Call contract.fundLoan() with value = amount in ETH
   - Return { success, txHash }

   - Call contract.repayLoan() with value
   - Return { success, txHash }

   - Call contract.getLoansByBorrower(address)
   - Format and return loan array

5. getAllPendingLoans(contract)
   - Call contract.getAllPendingLoans()
   - Return formatted array

6. getProtocolStats(contract)
   - Call contract.getProtocolStats()
   - Return { totalLoans, tvl }
